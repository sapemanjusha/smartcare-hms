# SmartCare HMS 🏥

Hospital Management System built using PHP & MySQL.

## Features
- Patient Management
- Doctor & Appointments
- Prescriptions
- Admissions & Discharge
- Billing & PDF Invoice
- Reports Dashboard
- Role-based Login System

## Setup Instructions

1. Clone the repo:
git clone https://github.com/sapemanjusha/smartcare-hms.git

2. Move to XAMPP:
C:\xampp\htdocs\smartcare_hms

3. Start Apache & MySQL

4. Import database:
- Open phpMyAdmin
- Create DB: smartcare_hms
- Import database/smartcare_hms.sql

5. Run:
http://localhost/smartcare_hms

## Default Login
Username: admin
Password: (generate using generate_password.php)
